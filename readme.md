# 🏏 IPL Cricket Data Analysis

## 📌 Overview
Complete Exploratory Data Analysis on 17 seasons
of IPL cricket data (2007/08 to 2024).

## 📊 Dataset
- **Source:** Kaggle IPL Complete Dataset
- **matches.csv:** 1,095 matches
- **deliveries.csv:** 260,920 deliveries

## 🔑 Key Findings
1. Toss winner wins only **50.6%** — myth busted!
2. Fielding first wins **52.8%** of matches
3. **CSK** most consistent team (58.2% win rate)
4. **Virat Kohli** = most runs (8,014)
5. **Chris Gayle** = most sixes (359)
6. **YS Chahal** = most wickets (213)
7. **AB de Villiers** = most Player of Match (25)
8. **KKR** = best chasing team
